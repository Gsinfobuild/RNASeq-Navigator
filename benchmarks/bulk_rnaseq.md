# Bulk RNA-seq Benchmark

## Objective

Curate **20 manually validated Bulk RNA-seq BioProjects** for benchmarking the RNASeq Navigator classifier and workflow recommendation engine.

---

## Progress

**Validated Projects:** 1 / 20

---

## Validation Criteria

A project is included only if:

- Publicly available.
- Clearly identified as Bulk RNA-seq.
- Metadata can be independently verified.
- Expected workflow is known.
- Successfully analyzed by RNASeq Navigator.
