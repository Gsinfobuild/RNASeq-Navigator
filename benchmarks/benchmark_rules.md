# RNASeq Navigator Benchmark Validation Rules

A BioProject may only be added if:

1. The sequencing study type is clearly identifiable.

2. The metadata is publicly available.

3. The expected study type can be independently verified.

4. The recommended primary workflow is known.

5. The BioProject has not already been added.

6. The accession is valid.

7. Validation is recorded as Manual.

8. Notes explain any unusual metadata.

Projects failing these criteria should not be added.
